3 プラグイン開発

2016年11月24日 ogiwara コメントをどうぞ

ここからは、私がまだプラグイン開発を始めたばかり、という事もあってかなりあやふやな説明です。ご了承ください

(pocketmineプラグインのみ開発したことがある方向けのページです。IDEの機能などについてはツッコミ所があるかもしれませんがご了承ください)

MiNETでプラグインを開発するには、C#の最低限の知識(オブジェクト指向など)と、pocketmineプラグイン開発の経験が必要です

What is good for Developers?

pocketmineでMiNETでプラグイン開発をすることのメリットは、

  • 先進的なプログラミング言語であるC#について学べる
  • パケットのやり取りがログに保存され、MCPEのパケットについて詳しくなれる
  • C# + Visual Studioの組み合わせにより、非常に生産性の高いコーディングが行える

などであると私は思います

では、実際にプラグインをつくってみましょう

Visual Studioを開き、メニューのファイル->新規作成->プロジェクトを選択してください

その後、Visual C# -> クラスライブラリを選択して、プラグイン名とプラグインを置くディレクトリを決め、OKボタンを押して下さい

プロジェクトの生成が終了したら、ソリューションエクスプローラー -> Propertiesをダブルクリックして下さい

その後、「対象のフレームワーク」から、.NET framework 4.6.1を選択してください

そしたら、メニューのツール-> nugetパッケージマネージャー -> ソリューションのNuGetパッケージの管理を選択し、検索ボックスにMiNETと入力し、それをインストールしてください

幾つか出てくるウインドウの内容をよく読み、インストールを完了させてください

これで、プラグインの開発の準備は整いました。では、試しに次のコードを入力してみましょう。

コードの編集が終わったら、Ctrl + S で保存し、上の開始ボタンを押してください。その後、無事にコンパイルが完了したら、bin/Debug内にプラグインが生成されているはずです。
